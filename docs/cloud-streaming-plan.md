# Cloud Streaming Plan

This file is kept as a compatibility entry point. The current public cloud demo guide is [cloud-demo.md](cloud-demo.md).

Cloud Unity RenderStreaming is an optional extension. It should not block the local phone-to-Unity portfolio demo.

Use runtime placeholders for all infrastructure values:

- `<cloud-ssh-host>`
- `<cloud-ssh-port>`
- `<cloud-user>`
- `<remote-root>`
- `<turn-public-ip>`
- `<turn-username>`
- `<turn-credential>`

Do not commit real SSH hosts, ports, passwords, TURN credentials, public IPs, private server paths, or local run logs.
